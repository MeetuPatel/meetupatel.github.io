K-pop sensation

This project was built as a part of the class Interactive Information Visualisation at Northwestern University. We analysed the Spotify Music Dataset to find out the backstory behind k-pop's popularity. We use a storytelling approach to present our information. The first visualisation is a bar race chart which shows the years k-pop became famous. Our next visualisation is also a bar race chart which provided information on the major k-pop groups. Lastly, our final visualisation answers the question of why k-pop is so popular. This project was built using d3.js library. Click on the link below to view the project.
